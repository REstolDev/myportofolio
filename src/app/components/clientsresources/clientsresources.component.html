Guide for Filling in Excel File for WooCommerce Import

To ensure a successful product import to your WooCommerce store, follow these guidelines and make sure to include at least the minimum required information in the corresponding cells:

  1. SKU (Stock Keeping Unit): Assign a unique identifier to each product. This field is mandatory and is auto-generated if absent.
  2. Product Name (Name): Provide a descriptive name for your product. This field is mandatory.
  3. Product Type (Type): Indicate the type of product, such as simple, variation, or virtual. Use valid values: simple, variable, grouped, external, variation, virtual, or downloadable.
  4. Regular Price (Regular Price): Set the regular price of the product.
  5. Categories (Categories): Assign at least one category to the product. Use > to represent hierarchy, e.g., "Category 1 > Category 2".
  6. Stock Status (In stock?): Indicate whether the product is in stock (1) or not (0).
  7. Short Description (Short Description): Provide a brief description of the product.
  8. Long Description (Description): Offer detailed information about the product.
  9. Publication Status (Published): Set the publication status of the product (1 for published, 0 for private, -1 for draft).
  10. Catalog Visibility (Visibility in catalog): Define the visibility of the product in the catalog (visible, catalog, search, or hidden).

These are the minimum required cells. Make sure to adhere to the data format and follow additional guidelines provided in the documentation. You can add more details as needed for your products, but these are the essentials for a basic and successful import.

--- Image Upload Guide: ---

To upload images, you need to put the URLs of the images in the "Images" column of your Excel file, separated by commas.

>> Single Image:

For a single image, you just need to provide the direct URL of that image in the "Images" column.

Example for a single image:

/wp-content/uploads/2024/product1.jpg

> Gallery of Images:

If you have a gallery of images for a product, you should list the URLs of all the images in the "Images" column, separated by commas. The first image in the list will be considered the featured image.

Example for a gallery of images:

/wp-content/uploads/2024/product2_1.jpg, /wp-content/uploads/2024/product2_2.jpg, /wp-content/uploads/2024/product2_3.jpg

Note:

  • Make sure to use the correct URLs for your images.
  • If you are creating a gallery, remember to separate the image URLs with commas.
  • The first image in the list will be treated as the featured image.
